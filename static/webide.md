![img](/help/help93.png)

![img](/help/help94.png)

![img](/help/help95.png)

![img](/help/help96.png)

不勾选保留索引，在终端中使用 git status 查看状态提示 changes to be committed （已放至暂存区，暂存区被重置）；勾选保留索引，在终端中使用 git status 命令查看状态提示 Changes not staged for commit（未放至暂存区，暂存区未被重置）

输入新分支名称，自动切换至新分支，且在新分支和原分支中均成功恢复代码

![img](/help/help186.png)

##### 1.21.4 编辑器-缩进风格

![img](/help/help195.png)

选择缩进风格，space 或 tab

![img](/help/help196.png)

缩进风格为 tab 时，键入 tab 再键入 delete ，每次删除一个 tab

缩进风格为 space 时，键入 tab 再键入 delete ，每次删除一个空格
